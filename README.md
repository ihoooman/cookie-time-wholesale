# Cookie Time

وب‌اپ فروش Cookie Time با رابط فارسی و راست‌چین، سبد خرید، ثبت سفارش در
دیتابیس و ارسال متن آماده به واتساپ.

## امکانات

- منوی واقعی محصولات با عکس، دسته‌بندی، قیمت و موجودی
- سبد خرید مهمان و ثبت سفارش بدون فرم ارسال
- لینک واتساپ با متن آماده سفارش
- پنل مدیریت محافظت‌شده برای محصولات، عکس‌ها، موجودی، سفارش‌ها و کد تخفیف
- گزارش فروش و محصولات پرفروش
- PWA قابل نصب با طراحی موبایل‌محور و Liquid Glass
- Cloudflare D1 برای داده‌ها و R2 برای تصاویر آپلودی

## اجرا

```bash
npm install
npm run dev
```

## اعتبارسنجی

```bash
npm run lint
npm run test
npm run db:generate
```

متغیر runtime به نام `ADMIN_EMAILS` باید شامل ایمیل مدیر یا مدیران و با
ویرگول جدا شده باشد. شماره مقصد واتساپ در `lib/whatsapp.ts` تعریف شده است.
